# 🚌 Bus Reservation System (Java + MongoDB)

A desktop-based Bus Reservation System built using Java Swing and MongoDB.

## 🔐 Features

- Secure login using BCrypt password hashing
- Unique email registration
- Bus search functionality
- Seat selection system
- Duplicate seat prevention (DB level unique index)
- Ticket confirmation screen

## 🛠 Tech Stack

- Java
- Swing (UI)
- MongoDB
- BCrypt (Password Security)
